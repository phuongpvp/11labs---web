Tổng quan

11LABS API cho phép bạn chuyển đổi văn bản thành giọng nói AI chất lượng cao một cách tự động. API sử dụng kiến trúc REST, trả về JSON, và hỗ trợ CORS.

Base URL: https://11labs.id.vn/api/external/v1/

Quy trình hoạt động

1
Gửi text
POST /tts.php
2
Nhận Job ID
Hệ thống xếp hàng
3
Kiểm tra
GET /status.php
4
Tải MP3 + SRT
download_url & srt_url

Xác thực

Mỗi request yêu cầu API Key. Có 2 cách truyền:

Cách Ví dụ
Query Parameter (đơn giản) ?api_key=PL_xxx
Header (bảo mật hơn) x-api-key: PL_xxx
Bảo mật: Không chia sẻ API Key. Nếu bị lộ, liên hệ admin để cấp key mới.

Bắt đầu nhanh

Tạo file MP3 từ văn bản chỉ với vài dòng code:

Python 
# pip install requests
import requests, time

API_KEY = "PL_your_api_key"
BASE = "https://11labs.id.vn/api/external/v1"

# Bước 1: Tạo job
res = requests.post(f"{BASE}/tts.php?api_key={API_KEY}", json={
    "text": "Xin chào, đây là giọng nói AI.",
    "voice_id": "21m00Tcm4TlvDq8ikWAM",
    "model_id": "eleven_multilingual_v2"
})
job_id = res.json()["job_id"]
print(f"Job created: {job_id}")

# Bước 2: Chờ xử lý
while True:
    status = requests.get(f"{BASE}/status.php?api_key={API_KEY}&job_id={job_id}").json()
    if status["status"] == "completed":
        # Bước 3: Tải file MP3
        mp3 = requests.get(status["download_url"])
        with open("output.mp3", "wb") as f:
            f.write(mp3.content)
        print("✅ Đã tải: output.mp3")
        # Tải file SRT (phụ đề)
        srt = requests.get(status["srt_url"])
        if srt.status_code == 200:
            with open("output.srt", "wb") as f:
                f.write(srt.content)
            print("✅ Đã tải: output.srt")
        break
    elif status["status"] == "failed":
        print("❌ Thất bại")
        break
    time.sleep(3)

Tạo giọng nói (TTS)

Gửi văn bản để chuyển thành giọng nói. Job sẽ được xếp hàng và xử lý bởi hệ thống.

POST /tts.php

Tham số (JSON Body)

Tên Kiểu Mô tả
text BẮT BUỘC string Văn bản cần đọc (tối đa 10,000 ký tự)
voice_id tuỳ chọn string ID giọng đọc. Mặc định: 21m00Tcm4TlvDq8ikWAM (Rachel)
model_id tuỳ chọn string Model AI. Mặc định: eleven_multilingual_v2
voice_settings tuỳ chọn object Tuỳ chỉnh: stability, similarity_boost, speed

Models có sẵn

Model ID Mô tả Ghi chú
eleven_v3 Eleven v3 (Mới nhất) 🌟 Chất lượng cao nhất, hỗ trợ đa ngôn ngữ
eleven_multilingual_v2 Đa ngôn ngữ v2 🇻🇳 Khuyên dùng cho Tiếng Việt
eleven_turbo_v2_5 Turbo v2.5 ⚡ Nhanh nhất
eleven_flash_v2_5 Flash v2.5 💰 Tiết kiệm nhất
eleven_monolingual_v1 Đơn ngôn ngữ v1 🇬🇧 Chỉ Tiếng Anh

Kiểm tra trạng thái Job

Kiểm tra tiến trình xử lý. Khi hoàn thành, response sẽ chứa link tải file MP3 và file SRT (phụ đề).

GET /status.php

Tham số

Tên Kiểu Mô tả
api_key BẮT BUỘC string API Key của bạn
job_id BẮT BUỘC string Mã job nhận từ endpoint TTS

Các trạng thái

Status Ý nghĩa
pending Đang chờ trong hàng đợi
processing Đang xử lý bởi worker
completed ✅ Hoàn thành — có download_url và srt_url
failed ❌ Thất bại

Response khi hoàn thành

{
  "status": "completed",
  "job_id": "A1B2C3D4",
  "created_at": "2026-03-04 07:00:00",
  "download_url": "https://11labs.id.vn/api/results/A1B2C3D4.mp3",
  "srt_url": "https://11labs.id.vn/api/results/A1B2C3D4.srt"
}
Mẹo: Poll endpoint này mỗi 3-5 giây cho đến khi status là completed hoặc failed. File MP3 sẽ tồn tại trên server trong 6 giờ.

Xử lý hàng loạt

Gửi nhiều đoạn text cùng lúc, hệ thống sẽ xử lý song song trên nhiều máy chủ.

Python — Batch Processing 
import requests, time, os

API_KEY = "PL_your_api_key"
BASE = "https://11labs.id.vn/api/external/v1"
VOICE = "21m00Tcm4TlvDq8ikWAM"

# Danh sách text (có thể đọc từ file .txt hoặc .csv)
texts = [
    "Đoạn 1: Chào mừng bạn đến với kênh...",
    "Đoạn 2: Hôm nay chúng ta sẽ...",
    "Đoạn 3: Cảm ơn các bạn đã theo dõi...",
]

# Bước 1: Gửi tất cả jobs
jobs = []
for i, text in enumerate(texts):
    res = requests.post(f"{BASE}/tts.php?api_key={API_KEY}", json={
        "text": text, "voice_id": VOICE
    }).json()
    if res.get("job_id"):
        jobs.append(res["job_id"])
        print(f"✅ [{i+1}/{len(texts)}] Job: {res['job_id']}")
    time.sleep(0.5)

# Bước 2: Chờ & tải từng file
os.makedirs("output", exist_ok=True)
for job_id in jobs:
    while True:
        s = requests.get(f"{BASE}/status.php?api_key={API_KEY}&job_id={job_id}").json()
        if s["status"] == "completed":
            mp3 = requests.get(s["download_url"]).content
            with open(f"output/{job_id}.mp3", "wb") as f:
                f.write(mp3)
            print(f"📥 Tải xong: {job_id}.mp3")
            break
        elif "failed" in s["status"]:
            print(f"❌ Lỗi: {job_id}")
            break
        time.sleep(3)

Mã lỗi

Tất cả response lỗi đều có dạng:

{ "error": "Mô tả lỗi" }
HTTP Code Ý nghĩa Cách xử lý
400 Dữ liệu không hợp lệ Kiểm tra JSON body, text không được rỗng
401 API Key sai hoặc hết hạn Kiểm tra lại API Key
403 Hết quota ký tự Liên hệ admin nạp thêm
404 Job không tồn tại Kiểm tra lại Job ID
405 Sai HTTP method TTS dùng POST, còn lại dùng GET
500 Lỗi server Thử lại sau vài giây
503 Hệ thống bận Tất cả worker đang bận, thử lại sau 10-30s
